# Singer Server

## Installation

```shell
go install github.com/singerapi/singer-server
```

## Usage

```shell
singer-server
```

Then you can log API Client at site [http://127.0.0.1:8080/singerapi/](http://127.0.0.1:8080/singerapi/)
